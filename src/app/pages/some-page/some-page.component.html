Some

El método some() itera el array y retorna un booleano si como mínimo uno de los elementos presentes en el array pasa una condición determinada. Recibe un callback que se encargara de preguntar aquello que queremos dentro del array.

Array original: {{ numbers() | json }}

Resultado al utilizar el método some en el cual le decimos que si alguno de los elementos del array son mayores de 2: {{ isGreaterThanTwo | json }}
